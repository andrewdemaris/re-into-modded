How to play: download pack.zip, import it into curseforge

How to edit: 
First time: Clone main, create a branch. 

You've already done this: Update your branch (pull from main). Don't need a new branch for every time.

Do your edits. Commit when you finish a task. 
When you're "Alright, I'm done for now" send a pull request. 
I'll review them, then you can respond to any comments, and at some point it'll get merged into main. 

Modpack files:
What's contained in the modpack.zip, but unzipped because these are what are intended to be edited.

Pack.zip - curseforge bundles a modlist and config overrides to let someone download the mods themselves.
